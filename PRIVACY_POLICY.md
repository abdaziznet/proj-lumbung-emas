## 1. INTRODUCTION

LumbungEmas ("We", "Us", "Our" or "App") adalah aplikasi mobile untuk manajemen portfolio logam mulia (emas & perak) dengan tracking keuntungan & kerugian real-time. Privacy policy ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi data Anda.

## 2. DATA WE COLLECT

### 2.1 Account & Authentication Data
Ketika Anda sign-in menggunakan Google Sign-In, kami menerima:
- Email address
- Display name
- Profile picture URL
- Google account ID

**Purpose:** Untuk identifikasi pengguna & personalisasi pengalaman

### 2.2 Portfolio Data
Data portfolio Anda disimpan di Google Sheets yang Anda kontrol:
- Asset details (gram, harga beli, tanggal beli)
- Transaction history
- Brand information (Antam, UBS, EmasKu, dll)
- Price data

**Purpose:** Untuk menghitung profit/loss & analytics

### 2.3 Price Data
Kami mengumpulkan harga logam mulia untuk perhitungan:
- Daily buy & sell prices
- Metal type (gold/silver)
- Brand information

**Purpose:** Untuk kalkulasi real-time portfolio value

### 2.4 Technical Data
- Device type & OS version
- App version & crash logs (jika aktif)
- Notification tokens
- Usage analytics (opsional)

**Purpose:** Untuk improve app stability & performance

### 2.5 Data Kami TIDAK Kumpulkan
- ❌ Location data
- ❌ Contacts atau address book
- ❌ Photos/media (tanpa permission)
- ❌ Biometric data
- ❌ Payment information

## 4. DATA STORAGE & SECURITY

### Storage Location
- **Authentication:** Firebase (Google Cloud)
- **Portfolio Data:** Google Sheets (Your Google Drive)
- **Local Cache:** SQLite (Device local storage)
- **Pricing:** In-memory cache

### Security Measures
- SSL/TLS encryption for all API calls
- OAuth 2.0 untuk authentication
- Local device encryption (via Android Keystore)
- No passwords stored (Google Sign-In only)
- HTTPS untuk semua external connections

### Data Retention
- Authentication data: Retained saat account aktif
- Portfolio data: Anda yang kontrol (di Google Sheets)
- Device data: Deleted saat uninstall atau clear app data
- Crash logs: Retained max 30 hari
